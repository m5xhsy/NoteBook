# 类和对象

> **C++面向对象三大特性：`封装`，`继承`,`多态`。**
>
> **C++认为`万事万物皆为对象`,对象上有其属性和行为**

**创建一个简单的对象示例：**

```c++
#include <iostream>
using namespace std;

const double PI = 3.14;

class Circle {
	// 访问权限
public:
	// 属性
	int r;
	// 行为
	double get_perimeter() {
		return 2 * PI * r;
	}
};


int main() {
	Circle c1;  // 实例化，通过类创建对象
	c1.r = 10;
	cout << "周长:" << c1.get_perimeter() << endl; // 周长:62.8
	return 0;
}
```
