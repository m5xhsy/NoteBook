## 常用限定符

| 代码/语法 | 说明             |
| --------- | ---------------- |
| *         | 匹配零次或更多次 |
| +         | 匹配一次或更多次 |
| ?         | 匹配零次或一次   |
| {n}       | 匹配n次          |
| {n,}      | 匹配n次或更多次  |
| {n,m}     | 匹配n到m次       |

## 常用反义词

|          |                                       |
| -------- | ------------------------------------- |
|          |                                       |
|          | 匹配任意不是空白符的字符              |
| \D       | 匹配任意非数字的字符                  |
| \B       | 匹配不是单词开头或结束的位置          |
| [^x]     | 匹配除了x以外的任意字符               |
| [^aeiou] | 匹配除了aeiou这几个字母以外的任意字符 |
